Crypto-currency Open Source RL Environment

Description

  • Dema Ushchapovskyy, Suraj Tirupati

  • Ongoing

Building an open source OpenAI Gym enviroment to allow people to experiment and deploy reinforcement learning agents on the crypto currency market.

Modular environment, which allows researchers and or developers to test reinforcement learning agents. The modularity of the environment allows the user to change the granularity of trading, to trade various multiple currencies at a time and engineer wrappers of the environment to come up with more complex strategies around trading. Support for multiple common currencies such as BTC, ETH and EOS. Multiple as well as single currencies can be traded.
