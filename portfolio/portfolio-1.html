Audio Source Separation

Description

  • Dema Ushchapovskyy, Suraj Tirupati

  • June, 2019

Built a system which extracts vocal information from songs. The approach used to solve this task was an autoencoder, which is trained with short time fourier transform of songs and their respective vocals.

Final result was 2db SNR on the vocal extracted. The progress on this project is still ongoing.

Technology

  • Python
  • Keras
  • MATLAB
