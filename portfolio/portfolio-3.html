OpenAI Gym with RL

Description

  • Dema Ushchapovskyy

  • Ongoing

Building reinforcement learning algorithms from the bottom up to solve various OpenAI gym environments.

Implemented algorithms such as DQN, DDQN, DRQN, DARQN, A2C, A3C, PPO all from scratch to improve my understanding of reinforcement learning. As well as that I experimented with extending Python with C in order to speed up various bottlenecks in the learning process.

Environments attempted so far include BlackJack, Pacman, Cartpole, BipedalWalker and others.
