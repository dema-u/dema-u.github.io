"Mars" Rover

Description

  • Dema Ushchapovskyy, Carlo Adrian, Maria Belwakowska,Yasir Soleja

  • March, 2017

First year project where the task was to build a rover which can navigate around remotely and scan information about various rocks, by scanning it's EM radiation spectrum.

My contribution to this project was programming an Arduino to control the servos and motors on the rover. As well as that I built a small website to send commands to the Arduino.

Final grade: 1st

Technology

  • C++
  • Arduino
  • Analogue Electronics
